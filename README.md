# Design and Analysis of Algorithms (DAA) Lab Programs

This repository contains the implementation of various Design and Analysis of Algorithms (DAA) laboratory programs in C. The programs demonstrate different algorithm design techniques and their practical applications.

## Programs Included

1. Merge Sort
2. Topological Sort
3. Element Uniqueness using Pre-Sort (Quick Sort)
4. Dijkstra's Algorithm
5. Floyd-Warshall Algorithm
6. Boyer-Moore String Matching
7. N-Queens Problem
8. Selection Sort Analysis
9. Tower of Hanoi

## Prerequisites

* GCC Compiler
* Linux/Unix Environment

## How to Compile and Run

Create a source file:

```bash
gedit file.c
```

Compile the program:

```bash
cc file.c
```

Run the executable:

```bash
./a.out
```

Alternatively:

```bash
gcc file.c -o output
./output
```
